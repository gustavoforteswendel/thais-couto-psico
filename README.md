# 🌿 Thais Couto | Psicóloga e Coach

Landing page institucional desenvolvida para apresentar os serviços da psicóloga **Thais Couto**, com foco em uma experiência moderna, acolhedora e minimalista.

O projeto foi criado utilizando **HTML, CSS e JavaScript puro**, incorporando animações suaves com **GSAP** e **ScrollTrigger** para proporcionar uma navegação fluida e transmitir a sensação de conforto e bem-estar alinhada à identidade visual da profissional.

## ✨ Destaques

- Design minimalista e elegante
- Motion design utilizando GSAP
- Animações baseadas em ScrollTrigger
- Hero section personalizada com árvore inspirada na identidade visual da marca
- Efeito contínuo de folhas flutuando em loop
- Layout totalmente responsivo
- Foco em acessibilidade e performance
- Paleta de cores inspirada na identidade da psicóloga
- Integração com WhatsApp e Doctoralia

## 🎨 Conceito visual

O projeto foi desenvolvido buscando transmitir:

- Acolhimento
- Confiança
- Leveza
- Humanização
- Equilíbrio
- Sofisticação discreta

A principal inspiração foi a própria identidade visual da profissional, utilizando a árvore presente em sua marca como elemento central do storytelling visual.

## 📱 Responsividade

A interface foi projetada para funcionar perfeitamente em:

- Desktop
- Notebook
- Tablet
- Smartphone

As animações também respeitam usuários que utilizam **`prefers-reduced-motion`**, reduzindo efeitos quando necessário.
